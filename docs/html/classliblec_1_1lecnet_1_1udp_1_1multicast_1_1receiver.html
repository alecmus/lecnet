Get UDP broadcast message obtained by the last call to run.

Parameters
messageThe message will be written here, if available. A maximum of 1024 characters will be acquired.
Parameters
errorError information.
Returns
Returns true if successful, else false.

A call to run, and then checking for the status of running is important for this call to make sense. After a successful call to run, wait until running returns false, then call this function. After a message is retrieved using this function it is immediately deleted from memory. A new call to run can then be used to check for a new message.
